Para utilizar api e necessario usar o comando: php index.php "cep que deseja ser consultado";
Ex: php index.php 20081240

Esse comando ira fornecer os seguintes resultados: 
Array
(
    [cep] => 20081-240
    [logradouro] => Praça Mauá
    [complemento] => lado ímpar
    [bairro] => Centro
    [localidade] => Rio de Janeiro
    [uf] => RJ
    [ibge] => 3304557
    [gia] => 
    [ddd] => 21
    [siafi] => 6001
)

com o app.php rodando, e digitar na barra de pesquisa localhost:3000/front.php
